# mapとpluck

## 概要

両者の違いメモ。

### ざっくり

| 項目 | map                    | pluck                    |
| ---- | ---------------------- | ------------------------ |
| 役割 | 形を変える             | 値を抜き出す             |
| 入力 | オブジェクトでもOK     | オブジェクトでもOK       |
| 出力 | 新しい構造のCollection | 指定キーだけのCollection |

## mapは変換

```php
<?php
collect([1, 2, 4])->map(fn ($n) => $n * 2);

// [2, 4, 6]
```

=> 1個1個を加工して別物にする

## pluckは切取

```php
<?php
collect([['id' => 1, 'name' => 'john', 'role' => 'admin'], ['id' => 2, 'name' => 'pyon', 'role' => 'rabbit']])->pluck('name')

= Illuminate\Support\Collection {#7988
    all: [
      "john",
      "pyon",
    ],
  }
```

=> nameだけを取り出している

```php
<?php
collect([['id' => 1, 'name' => 'john', 'role' => 'admin'], ['id' => 2, 'name' => 'pyon', 'role' => 'rabbit']])->pluck('role', 'name')

= Illuminate\Support\Collection {#7971
    all: [
      "john" => "admin",
      "pyon" => "rabbit",
    ],
  }
```

=> 第二引数に`キー`を指定すると連想配列の形で抽出することもできる
