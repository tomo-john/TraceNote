# Query Builder(クエリビルダ)

## 概要

クエリビルダの基本確認

### クエリビルダとは

> SQLを組み立てるためのオブジェクト。

```php
<?php
$query = Trace::query();
```

=> この時点ではまだDBに問い合わせていない(データ持ってない)

=> これからどんなSQLを発行するかを覚えているだけ

### get()で初めて実行

```php
<?php
$trace = Trace::query()
    ->where('status', 'draft')
    ->latest() // ここではまだSQL発行されていない
    ->get();   // ここでSQLが実行される
```

=> `get()`なら返ってくるものは`Collection`

### Collectionとの違い

クエリビルダでできること:
- where()
- orderBy()
- latest()
- with()
- paginate()

など、DBから何を取るのかを決める。

Collection:
- map()
- filter()
- groupBy()

など、取得したデータを加工する。
