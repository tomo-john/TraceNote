# 期間の枠作る検証

## 概要

コレクション、配列操作をtinkerで確認

### 1. 期間の枠の作り方

```php
<?php
collect(range(0, 4))->map(fn ($days) => now()->subDays($days)->format('Y-m-d'))->toArray();

= [
    "2026-06-12",
    "2026-06-11",
    "2026-06-10",
    "2026-06-09",
    "2026-06-08",
  ]
```

## 2. 連想配列を作る

```php
<?php
collect(range(0,4))->mapWithKeys(function ($days) { $date = now()->subDays($days)->format('Y-m-d'); return [$date => 0]; })->toArray();

= [
    "2026-06-12" => 0,
    "2026-06-11" => 0,
    "2026-06-10" => 0,
    "2026-06-09" => 0,
    "2026-06-08" => 0,
  ]

collect(range(0, 4))->mapWithKeys(function ($days) {
    $date = now()->subDays($days)->format('Y-m-d');
    
    // 💡 [ キー => 値 ] の形（連想配列）で返すのがルール！
    return [$date => 0];
})->toArray();
```

- Collection同士で実施して最後に配列を返す
- 連想配列のキー(今回日付)が同じだったら、後から入ってきた値で上書きする

### merge()

| 左側(ベース) | 右側(引数) | 動く？ |
| ------------ | ---------- | ------ |
| Collection   | Collection | OK     |
| Collection   | Array      | OK     |
| Array        | Collection | NG     |

- 引数ArrayはLaravelが裏で配列をコレクション化してくれる
- 普通の配列は`merge()`メソッドを持たないためNG
