# Collection

## 概要

Collectionと配列について

### 配列(array)とは

PHPにもともとあるデータ構造。

```php
<?php
$numbers = [1, 2, 3];
```

連想配列も配列。

```php
<?php
$user = []
    'name' => 'john',
    'age' => '10',
;
```

特徴:
- PHP標準機能
- 軽量
- foreachできる
- メソッドは持っていない

`[1, 2, 3] => map(...)`はエラーになる。

### Collectionとは

Laravelが用意してくれている配列を便利に扱うためのクラス。

```php
<?php
$numbers = collect([1, 2, 3]);

// => Illuminate\Support\Collection というオブジェクト
```

```bash
# イメージ
Query Builder -> get() -> Eloquent Collection -> 配列
```
