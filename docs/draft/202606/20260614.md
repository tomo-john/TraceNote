# mapWithKeys()

## 概要

キーも自分で決めて配列を作る。

### 普通のmap()

```php
<?php
collect([1, 2, 3])
    ->map(fn ($n) => $n * 10)
    ->toArray();

// [10, 20, 30]
```

=> 要素を変換するだけ

### mapWithKeys()
```php
<?php
collect([1, 2, 3])
    ->mapWithKeys(fn ($n) => [
        $n => $n * 10
    ])->toArray();

[
    1 => 10,
    2 => 20,
    3 => 30,
]
```

- コールバックの中では、1つの連想配列を返す必要がある
- しかしあくまで戻り値はCollection
