# tinker

## 概要

Laravelに標準搭載されているREPL(Read-Eval-Print Loop)ツール。

## 起動

```
php artisan tinker
```

## compact()

PHPの標準関数。

すでにある変数名と同じ名前のキーを持った連想配列を、自動でがっちゃんこして作ってくれる関数。

```php
<?php
$language = 'PHP'
$framework = 'Laravel'

compact('language', 'framework')

= [
    "language" => "PHP",
    "framework" => "Laravel",
  ]
```

引数には変数そのものではなく、`変数名の文字列`を渡す。


## view()の第二引数

第二引数には、ビュー側に渡したいデータを指定する。

このとき、第二引数には連想配列を渡すのが基本。

=> 配列のキーがビューで使う変数名、値がその変数の中身になる。

```php
<?php
return view('sandbox', compact('language', 'framework'))

// Blade側
{{ $language }}  // PHP
{{ $framework }} // Laravel
```
