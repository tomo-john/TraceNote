# view()のデータ渡し

## 概要

配列を渡すときの注意点など

### 基礎

`view()`の`view('ビュー名', 配列)`は渡された配列のキー名をそのまま変数名に展開する仕組み。

例えば、渡す予定の`$stats`が以下のような連想配列:

```php
$stats = [
    'traceCount' => 10,
    'tagCount' => 5,
];
```

これを、`view('dashboard', $status)`みたいに渡すと、Blade側では`$stats`という変数でなく、

`$traceCount`や`$tagCount`という独立した変数として展開される。

なので、Blade側で`{{ $stats['traceCount'] }}`と書くと、`$stats`なんて知らないと起こられる。
