# partialとBlade Component

## 概要

両者の違いなどおさらい。

## partial

> Bladeを分割するためのファイル

```bash
partials/
    relation-card.blade.php
    add-relationo-modal.blade.php
```

みたいに分ける。

呼び出しは:

```blade
@include('livewire.trace.partisals.relation-card')
```

=> コードを読みやすくするための分割

### partisalの特徴

- Bladeだけ
- PHPクラスなし
- 状態を持たない
- 単なるテンプレートの切り出し

## Blade Component

> 再利用できる部品

- 部品として再利用
- propsを受け取れる
- 専用クラスも作れる
- 他画面でも使える

=> UI部品

まず、`partial`。必要になったら`Component`。
