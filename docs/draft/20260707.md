# Bladeコンポーネント

## 概要

UIの部品

## 2種類

| 種類      | ファイル    | PHPクラス | 用途               |
| --------- | ----------- | --------- | ------------------ |
| Anonymous | Bladeのみ   | なし      | 見た目だけの部品   |
| Class     | Blade + PHP | あり      | ロジックを持つ部品 |

### Anonymous Component（匿名コンポーネント）

```bash
resources/views/components/button.blade.php
```

```blade
<button {{ $attributes }}>
    {{ $slot }}
</button>
```

```blade
<x-button>
    保存
</x-button>
```

見た目だけを共通化したいならこれでOK => 共通のボタンなど


## Props

コンポーネントに渡す、`設定値`や`引数のようなもの`

```blade
@props(['type'])
```

=> このコンポーネントは`type`というPropsを受け取りますという宣言

=> 受け取った後は普通の変数として使用できる

デフォルト値も設定できる:

```blade
@props([
    'type' => 'button'
])
```

### step1: 作ってみる

`resources/views/components/ui/button.blade.php`:

```blade
@props([
    'type' => 'primary',
])

<button>
    {{ $slot }}
</button>
```

呼び出し:

```blade
<x-ui.button>DOG</x-ui.button>
```

=> `{{ $slot }}`はDOGが入る場所

### step2: Propsを追加する

```blade
@props([
    'type' => 'primary',
])

@php
    $classes = match($type) {
        'primary' => 'bg-blue-100 text-blue-500',
        'danger' => 'bg-red-100 text-red-500',
        default => 'bg-gray-100 text-black',
    };
@endphp

<button class="{{ $classes }} px-2 py-1 rounded-xl">
    {{ $slot }}
</button>
```

```blade
<flex class="gap-2">
    <x-ui.button>保存</x-ui.button>
    <x-ui.button type="danger">削除</x-ui.button>
    <x-ui.button type="dog">DOG</x-ui.button>
</flex>
```
