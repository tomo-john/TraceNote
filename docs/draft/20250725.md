# バリエーション

## 概要

`rules()`は自動で呼ばれる。

## rules()

```php
<?php
protected function rules(): array
{
    return [
        'name' => ['required'],
        'email' => ['required'],
    ];
}

public function save()
{
    $this->validate();
}
```

`save()`の中では`$this->validate()`しか書いていない。

なのに、`rules()`が使われている。

これはLivewire(Laravel)が内部で、`$this->rules()`を呼んでくれているため。

## 名前を変えた場合

```php
<?php
protected function profileRules(): array
{
    return [
        ...
    ];
}
```

この場合は、`$this->validate();`だと`rules()`がない！となってしまうので

```php
<?php
$this->validate($this->profileRules());
```

のように、自分でルールを渡す必要あり。
