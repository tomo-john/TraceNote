# relatedTraces()

## 概要

関連するTraceを取得するメソッドについて考える。

## 初期作成

単純にincoming, outgoingで取得してmergeする

```php
<?php
    public function relatedTraces()
    {
        $incoming = $this->incomingRelations()
                         ->where('relation_type', 'related')
                         ->with('fromTrace')
                         ->get()
                         ->pluck('fromTrace');

        $outgoing = $this->outgoingRelations()
                         ->where('relation_type', 'related')
                         ->with('toTrace')
                         ->get()
                         ->pluck('toTrace');

        return $incoming->merge($outgoing);
    }
```

この場合、同じTraceが2回入る場合がある。

| from | to | type    |
| ---- | -- | ------- |
| 1    | 2  | related |
| 2    | 1  | related |

## ユニーク 

```php
<?php
return $incoming->merge($outgoing)
                ->unique('id')
                ->values();
```

### unique()

重複した要素を取り除いてくれるメソッド。

```php
<?php
$collection = collect([1, 2, 2, 3, 3, 3]);
$collection->unique();
// [1, 2, 3]
```

`unique()`は要素を削除しても、元のキーはそのまま残すので注意点。

=> 次の`values()`でキーを振り直す

### values()

キーを0から振り直してくれるメソッド。

```php
<?php
// values()なし
collect([1,2,1,3,4,1,2])->unique()

= Illuminate\Support\Collection {#8921
    all: [
      0 => 1,
      1 => 2,
      3 => 3,
      4 => 4,
    ],
  }

// values()あり
collect([1,2,1,3,4,1,2])->unique()->values()

= Illuminate\Support\Collection {#8957
    all: [
      1,
      2,
      3,
      4,
    ],
  }
```
