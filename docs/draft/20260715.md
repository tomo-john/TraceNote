# 観察コマンド

## 概要

何かをインストールした時に確認する用のコマンド。

## CMD

```bash
systemctl status <サービス名>
```

=> 動いてる？

```bash
sudo ss -tlnp
```

=> どのポート開いた？

```bash
ps aux | grep <サービス名>
```

=> プロセスはある？

```bash
ls /etc/<サービス名>
```

=> 設定ファイルはどこ？

## ss

ソケット(Socket)の状態を表示するコマンド。

`netstat`の後続として使用されることが多く、高速に動作する。

```bash
ss -tlnp
```

現在待ち受け(LISTEN)しているTCPポートと、そのプロセスを確認するときに使用。

- -t: TCPソケットのみ表示
- -l: LISTEN状態のみ表示
- -n: ポート番号・IPアドレスを名前解決しない(高速)
- -p: ソケットを使用しているプロセス情報(PID・プロセス名)も表示

```bash
ss -tlnp

State   Recv-Q Send-Q Local Address:Port  Peer Address:Port Process
LISTEN  0      511    0.0.0.0:80         0.0.0.0:*        users:(("nginx",pid=1200,fd=6))
LISTEN  0      128    127.0.0.1:3306     0.0.0.0:*        users:(("mysqld",pid=980,fd=20))
LISTEN  0      128    0.0.0.0:22         0.0.0.0:*        users:(("sshd",pid=765,fd=3))
```

`0.0.0.0:80`: すべてのNICで80番ポートを待ち受けている

`127.0.0.1:3306`: ローカルホストからしか接続できない
