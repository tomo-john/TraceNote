# wire:click

## 概要

Livewireのwire:click()の挙動

## 基本

`wire:click`はLivewireコンポーネント(PHPクラス)のpublicメソッドを実行するためのもの。

例えば:

```php
<?php
use Livewire\Component;

class Counter extends Component
{
    public $count = 0;

    public function increment()
    {
        $this->count++;
    }
}
```

```blade
<button wire:click="increment">
    +1
</button>
```

ボタンを押すと:

```
ブラウザ -> Livewire -> Counter::increment() -> render() -> 画面更新
```

## 呼べないもの

- private
- protected
- Blade内のPHP関数
- JavaScritp関数

## 他クラスのメソッドは？

```php
<?php
class DogServece
{
    public function baw()
    {
        ...
    }
}
```

```blade
wire:click="baw()"
```

では呼ぶことができない。

=> Livewireのpublicメソッドを入口にして、その中でサービスクラスを呼び出す形ならOK

## まとめ

> wire:clickで直接呼び出せるのは、Livewireコンポーネントに定義されたpublicメソッドだけ。
> そのpublicメソッドの中では、通常のLaravel/PHPコードとして、自作クラスやServiceクラスなど何でも呼び出せる。
