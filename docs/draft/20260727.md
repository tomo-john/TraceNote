# app/Providers/AppServiceProvider.php

## 概要

アプリ起動時の初期設定を書く場所。

## app/Providersって？

```bash
Laravel起動
    |
    ServiceProviderを順番に実行
    |
    |- AppServiceProvier
    |- FortifyServiceProvider.php
    |- ...
```

## AppServiceProvider

```php
<?php

    // 起動時に呼ばれているメソッド
    protected function configureDefaults(): void
    {
        Date::use(CarbonImmutable::class);

        DB::prohibitDestructiveCommands(
            app()->isProduction(),
        );

        Password::defaults(fn (): ?Password => app()->isProduction()
            ? Password::min(12)
                ->mixedCase()
                ->letters()
                ->numbers()
                ->symbols()
                ->uncompromised()
            : null,
        );
    }
```

## Password::defaults

### fn() => ...

アロー関数(無名関数): `function () { return ...; }`とほぼ同じ意味。

defaults()の引数に関数を渡しているので、これがデフォルトの関数となる。

### : ?Password

この関数の戻り値はPassword(オブジェクト)またはnullですの宣言。

### app()->isProduction()

`app()`はLaravelアプリケーションそのものを返すヘルパー関数。

`app()->isProduction()`は`APP_ENV=local`ならfalseを返す。

=> 今、本番環境で動いてる？を聞いてる関数

つまり、本番環境かそうでないかで、Password::defaults()に渡す関数の内容が変わる。

=> 開発環境なら`null`

### 開発環境の時のPasswordオブジェクト

```php
<?php
Password::defaults(fn (): ?Password => app()->isProduction()
    ? Password::min(12)
        ->mixedCase()
        ->letters()
        ->numbers()
        ->symbols()
        ->uncompromised()
    : null,
);
```

これは、`/vendor/laravel/framework/src/Illuminate/Validation/Rules/Password.php`を見る。

`Password::min(12)`:

```php
<?php
// $minの初期値
protected $min = 8;

// new Password(12)される
public static function min($size)
{
    return new static($size);
}

// new されたときに実行
public function __construct($min)
{
    $this->min = max((int) $min, 1);
}
```

なので、`$this->min = 12;`になる。

`->mixedCase()`

```php
<?php
// 初期値
protected $mixedCase = false;

public function mixedCase()
{
    $this->mixedCase = true;

    return $this;
}
```

`$this->mixedCase = true`をセットしているだけ。

`->letters(), ->numbers()`も同様にプロパティに`true`をセットしている。

最後に全部まとめると...

本番環境では、

```bash
1. 最低12文字
2. 大文字・小文字を含む
3. 英字を含む
4. 数字を含む
5. 記号を含む
6. 漏洩済みパスワードではない
```

という設定が1つのPasswordオブジェクトに積み重なっている。
