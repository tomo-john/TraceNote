# OpenSSH

## 概要

Opne Secure Shellはネットワーク越しに別のコンピュータへ安全に接続するためのソフトウェア。

## できること

- リモートログイン(SSH)
- ファイル転送(scp)
- ディレクトリごとコピー(rsync + SSH)

## SSHクライアント

> 接続する側

コマンド:

```bash
ssh
scp
sftp
ssh-keygen
```

## SSHサーバー(sshd)

サービス名は`sshd`

```bash
# 起動確認
sudo systemctl status sshd

# 起動
sudo systemctl start sshd

# 自動起動
sudo systemctl enable sshd
```

## SSHの鍵

OpenSSHで作る鍵は`ssh-keygen`で作成するもの。

目的は、サーバーへ本人であることを証明するため。

```
PC
 |__ 秘密鍵(自分だけ)
 |__ 公開鍵(サーバーへ登録)
```

作成:

```bash
ssh-keygen -t ed25519 -C "john@example.com"
```

- `-t`: 鍵の種類。ed25519: 現在推奨される方式。
- `-C`: コメント

```bash
# 実行すると
Generating public/private ed25519 key pair.

# 次に
Enter file in which to save the key

# 普通はそのままENter
~/.ssh/id_ed25519

# パスフレーズ
Enter passphrase:
# => 設定すると秘密鍵 + パスフレーズの二段階になるので安全
```

```bash
# ここにできる
~/.ssh/
    |__ id_ed25519      # 秘密鍵
    |__ id_ed25519.pub  # 公開鍵
```
