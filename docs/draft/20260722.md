# vsftpd

## 概要

Very Secure FTP Daemon

Linuxでよく使われるFTPサーバーのソフトウェア。

## FTPおさらい

FTP(File Transfer Protocol)はファイルを送受信するための通信プロトコル。

=> ファイルのアップロードやダウンロードなど

## vsftpdとは？

FTPを利用できるようにするサーバープログラム。

## 設定ファイル

`/etc/vsftpd.conf`で主に行う。

```bash
# 匿名ログイン禁止
anonymous_enable=NO

# Linuxユーザーでログイン可能
local_enable=YES

# アップロード可能
write_enable=YES

# 作成されるファイルのパーミッション設定
local_umask=022
```

## ユーザー・パスは？

vsftpd自身はユーザー情報を持っていないことが多い。

通常はLinuxのユーザーアカウントをそのまま使う。

## FTPログイン後のディレクトリなどは？

デフォルトでは、`/home/john`などのユーザーのホームディレクトリ。

設定ファイルで
```bash
chroot_local_user=YES
```

これを設定すると、`/home/john`がFTPから見ると`/`のように見えたりする。

`sudo usermod -d /data/ftp/john john`のようにユーザーのホームディレクトリを変更なども。

または、`local_root=/data/ftp`のように設定ファイルで開始ディレクトリを変更することも。

## 専用ユーザー

一般的には、SSHなどでログインして作業する一発ユーザーとは別に、FTP専用のユーザーを作成したりする。

FTP専用ユーザーでのSSHログインを禁止したりすることも多い。
