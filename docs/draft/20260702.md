# Factory作成

## 概要

Traceのテスト用データを作成する

## 今回作成したいもの

タイトルなど固定のものを3件ほど。

- `database/factories/TraceFactory.php`を作る
- `database/seeders/DatabaseSeeder.php`から呼び出す

## Factory作成コマンド

```bash
# Traceモデルに対応するFactoryを作る(今回はこれ)
php artisan make:factory TraceFactory
```

LaravelはFactory名から対応するモデルを推測してくれるので、

`App\Models\Trace`が存在していれば自動で、`protected $model = Trace::class;`してくれる。

=> Laravel11以降であれば、PHPDocのジェネリクスでモデルとの対応を表現

Factory名とモデル名が一致しない場合や、明示したい場合はこれ。

```bash
php artisan make:factory TraceFactory --model=Trace

php artisan make:factory TraceFactory -m Trace
```

モデル作成時に、Factoryを作るなれこれ。

```bash
php artisan make:model Trace --factory
```

## TraceモデルにFactoryの追加

```php
<?php
// Factoryクラスをインポート
use Illuminate\Database\Eloquent\Factories\HasFactory;

class Trace extends Model
{
    // クラスの直下で、HasFactoryトレイトをuse
    use HasFactory;
    ...
```

## TraceFactory.php

初回イメージ:

```php
<?php

namespace Database\Factories;

use App\Models\Trace;
use App\Models\User;
use App\Enums\TraceStatus;
use Illuminate\Database\Eloquent\Factories\Factory;

/**
 * @extends Factory<Trace>
 */
class TraceFactory extends Factory
{
    /**
     * Define the model's default state.
     *
     * @return array<string, mixed>
     */
    public function definition(): array
    {
        return [
            'user_id' => User::find(1),
            'title' => 'タイトル',
            'summary' => '概要',
            'content' => '本文',
            'status' => TraceStatus::DRAFT,
        ];
    }
}
```

## DatabaseSeeder.php

```php
<?php
use App\Models\Trace; // 追加

class DatabaseSeeder extends Seeder
{
    public function run(): void
    {
        // Userを先に作成しておく
        User::factory()->create([
            'name' => 'john',
            'email' => 'john@gmail.com',
            'password' => Hash::make('john1234'),
        ]);

        Trace::factory()->create(); // デフォルト1件

        $user = User::find(1);

        Trace::factory()->create([  // 自作1件
            'title' => 'PHP Array',
            'summary' => '2',
        ]);

        Trace::factory()->create([  // 自作1件
            'title' => 'Collection',
            'summary' => '3',
        ]);

        Trace::factory()->create([  // 自作1件
            'title' => 'map()',
            'summary' => '4',
        ]);

    }
}
```

Factoryにデフォルトの型を書き、DataSeeder側で`create([...])`に配列を渡して個別に上書きする。

=> 今回は`title`と`summary`のみオーバーライド

## Seeder実行

```bash
# DBを一度綺麗にリセットして、Seederを実行
php artisan migrate:fresh --seed
```
