# Secure Boot

## 概要

ディストリビューションが信頼している鍵で署名されたモジュールしかロードできない。

## 流れ

- 1. 秘密鍵・公開鍵作成
- 2. 公開鍵(MOK)を登録
- 3. モジュールを秘密鍵で署名
- 4. Secure Boot下でもロードできる

## Ubuntu系のサンプル

### 鍵作成

```bash
openssl req -new -x509 -newkey rsa:2048 \
-keyout MOK.priv \
-out MOK.pem \
-nodes \
-days 36500 \
-subj "/CN=My Module Signing/"
```

### 公開鍵登録

```bash
sudo mokutil --import MOK.pem
```

再起動して、MOK Managerで登録。

### 署名

```bash
/scripts/sign-file sha256 \
MOK.priv \
MOK.pem \
vmmon.ko
```
