# match()

## 概要

PHP 8.0より導入。`switch`との違いなど。

## 基本

`switch`(従来からある構文):

```php
<?php
$role = 'admin';

switch ($role) {
    case 'admin':
        $permission = 'full_access';
        break;
    case 'editor':
        $permission = 'edit_access';
        break;
    default:
        $permission = 'read_only';
}
```

`match`(PHP 8.0～):

```php
<?php
$role = 'admin';

$permission = match ($role) {
    'admin' => 'full_access',
    'editor' => 'edit_access',
    default => 'read_only',
};
```
