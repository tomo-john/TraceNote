# fixed

## 概要

CSSの`position: fixed;`

要素を画面に固定して、スクロールしても動かないようにする。

## 用途

- 常に表示したいヘッダーやナビゲーションバー
- モーダルやオーバーレイの背景

=> `fixed`はブラウザの画面(viewport)が基準

## 位置の指定

`fixed`単体では位置が決まらないので、以下のクラスをセットで使用。

| クラス                        | 意味                           |
| ----------------------------- | ------------------------------ |
| top-0 right-0 bottom-0 left-0 | 上下左右の位置(0=端にぴったり) |
| inset-0                       | 上下左右すべて0                |
| inset-x-0                     | 左右だけ0                      |
| z-50                          | 重なり順(z-index)              |

## ドロップダウンなら absolute が定番？

```
relative -> 親
    |
absolute -> メニュ
```

みたいな構造。

=> absoluteの位置は親のrelativeが基準になる
