# composer update

## 概要

Laravel本体やインストールされているパッケージのバージョンを更新するコマンド。

## 何をするのか？

- composer install

    `composer.lock`に書かれているバージョンをそのままインストール

- composer update

    `composer.json`の条件に合う最新バージョンを探して更新

    `composer.lock`を書き換える

## Composerは何を見ている？

```bash
"require": {
    "php": "^8.3",
    "laravel/framework": "^12.0"
}
```

Composerは今のPHPで動くパッケージを探そうと考える。

現在のPCのPHPバージョン -> Composer -> インストールするライブラリを決定。

```bash
PHP8.4 -> package A 5.0

PHP8,3 -> package B 4.8
```

みたいにPCによって同じ`composer.json`でも違うライブライが入ることがある。

## platformとは？

```bash
"platform": {
    "php": "8.3.6"
}
```

=> 今のPHPが何であっても`PHP8.3.6`として考えてねという命令。

## composer.json編集後

composer updateを実施する。

これで、変更した`composer.json`を元に新しい`composer.lock`が生成される。
