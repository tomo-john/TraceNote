# VPS

## 概要

Virtual Private Server : 仮想専用サーバー

=> 自分専用のレンタルサーバー

## ConoHa VPS

GMOインタねーネットグループが提供している、日本で人気のあるVPSサービス。

個人開発から企業まで幅広く利用されている。

## テスト

- サービス: VPS
- OS: Ubuntu 24.04(x86_64)
- 料金タイプ: 6ヶ月
- プラン: 2GB
- ネームタグ: ubuntu-lab

## イメージ

```
自宅PC
  |
インターネット
  |
ConoHaのデータセンター
  |
契約したVPS(Ubuntu 24.04)
```

物理的にはConoHaのデータセンターにあるけど、自分専用のUbuntuマシン。

## セキュリティグループ

ConoHaの管理画面からセキュリティグループをdefault(初期値)から`IPv4v6-SSH`へ変更。

初期値の`default`は外部からの通信を遮断するらしい...

```bash
nc -vz 160.251.234.145 22
Connection to 160.251.234.145 port 22 [tcp/ssh] succeeded!
```

- -v: 詳細なメッセージ表示
- -z: データを送受信せず、接続できるかだけを確認
