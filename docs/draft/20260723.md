# type="submit"

## 概要

HTML基礎おさらい

## 普通のHTML

```html
<form>
    <input type="text">

    <button>
        保存
    </button>
</form>
```

このボタンは、何も書かなくても`type="submit"`として扱われる。

## submitって？

> submit = フォームを送信する

```html
<form action="/user" method="POST">

    ...

    <button type="submit">
        保存
    </button>

</form>
```

保存ボタンを押すと、`POST /user`が送られる。

## Livewireになると

```blade
<form wire:submit="save">
```

を書くと、HTMLのsubmitイベントを横取りする。

本来、POST送信されるところをLivewireが`save()`を呼び出す。

```blade
<form wire:submit="save">

    <button type="submit">
        保存
    </button>

</form>
```

=> submit発生 -> Livewireが受け取る -> save() 実行

## wire:clickは？

```blade
<button wire:click="save">保存</button>
```

クリックされたら、`save()`を呼ぶだけ。(単純)

=> formとは関係ない

- `wire:submit` = フォーム全体が送信されたら、このメソッドを呼ぶ
- `wire:click` = このボタン(要素)がクリックされたらメソッドを呼ぶ
