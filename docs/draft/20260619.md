# Zone.Identifier

## 概要

Windowsでこのファイルはどこからきたかを記録するための仕組み。

正式には、`Mark of the Web (MOTW)`と呼ばれる。

### いつできる？

- ブラウザからダウンロードしたファイル
- Windows側で作成したファイル
- Windows → WSL(Ubuntu) にコピーしたファイル

などに対して、

```
20260617_er_diagram.png
20260617_er_diagram.png:Zone.Identifier
```

のようなファイルが作られる。

### 中身は？

```
[ZoneTransfer]
ZoneId=3
```

のような情報が入っていて、

- 0 : ローカルコンピュータ
- 1 : ローカルイントラネット
- 2 : 信頼済みサイト
- 3 : インターネット
- 4 : 制限付きサイト

を表している。

Linux側から見ると単なる`xxx.png:Zone.Identifier`という変なファイルに見える。

=> 消しても問題ないことがほとんど

### 対策？

Windows側で、画像ファイルのプロパティ -> セキュリティのとこの`許可する`にチェックを入れる。
