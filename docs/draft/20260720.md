# Node.js

## 概要

JavaScriptをブラウザ以外でも動かすための実行環境。

## 例

```bash
console.log("Hello");
```

これを`node test.js`で実行できる。

- PHP: `php artisan`
- JS: `node`

## npmとは？

PHPでいう`Composer`に近い存在。

どちらもライブラリを管理するツール。

| PHP           | Node.js      |
| ------------- | ------------ |
| Composer      | npm          |
| composer.json | package.json |
| vendor        | node_modules |

## package.jsonとは？

PHPでいう`composer.json`と同じ役割。

- 必要なライブラリ
- コマンド
- バージョン

## node_modulesとは？

Composerなら`vendor`、Nodeなら`node_modules`。

ここに全部インストールされる。

## nvmとは？

> Node Version Manager

Node.jsのバージョン管理ツール。

```
nvm use 18
nvm use 22
```

みたいに切り替える。

## npm installとは？

Laravelをクローンしたら

```bash
composer install
```

するのを、Node側では、

```bash
npm install
```

=> `package.json`を読んで、`node modules`が作られる。
