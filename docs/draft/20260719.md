# Composer

## 概要

PHPのパッケージマネージャー。

## 基本

PHPのライブラリをインストール・更新・管理してくれるツール。

LaravelやLivewireなどをインストールしたりできる。

## npmとの違い

| PHP           | JavaScript   |
| ------------- | ------------ |
| Composer      | npm          |
| Packagist     | npm Registry |
| composer.json | package.json |

```bash
PHP -> Composer
JavaScritp -> npm
```

## composer.json

```bash
{
    "require": {
        "php": "^8.2",
        "laravel/fortify": "^1.30",
        "laravel/framework": "^12.0",
        "laravel/tinker": "^2.10.1",
        "livewire/flux": "^2.9.0",
        "livewire/livewire": "^4.0",
        "livewire/volt": "^1.10"
    },
```

このアプリにはこれらのライブラリが必要ですという設計図。

=> まだライブラリ本体はない

## composer.lock

これは実際にインストールされた完成リスト。

インストールしたバージョンなどの記載もあり。

## installとupdate

`composer install`は`composer.lock`を信じる。

つまり、lockに書かれたバージョンをそのまま入れる。

`composer update`は、`composer.json`を見て最新版を探し取得。

=> `composer.lock`も更新する

## vendor

Composerがライブラリをインストールすると、`vendor/`ディレクトリが作成される。

インストールされたライブラリはここに全部入る。
