# $attributes->class() 

## 概要

BladeコンポーネントでCSSクラスを結合する

## 基本

Bladeコンポーネント:

```blade
<div {{ $attributes->class(['p-4', 'bg-white']) }}>
    {{ $slot }}
</div>
```

このコンポーネントを呼び出す時に:

```blade
<x-card class="rounded-lg shadow" />
```

と書くと、最終的に:

```blade
<div class="p-4 bg-white rounded-lg shadow">
```

- コンポーネント側のクラス
- 呼び出し側で渡したクラス

この2つを自動でマージしてくれる。


## 条件付きクラス

```blade
<button
    {{ $attributes->class([
        'px-4 py-2',
        'bg-blue-500' => $primary,
        'bg-gray-500' => !$primary,
    ]) }}
>
    ボタン
</button>
```

配列の`'クラス名' => 条件`という書き方がポイント。
