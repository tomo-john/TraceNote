# クエリビルダ

## when()

- 第1引数が`true`(データがある)のときだけ、第2引数の処理を実行
- `when($foo, fn ($query) => ...`

## where()

- 渡す引数によって2つの役割を行う

1. 通常の`where`(条件を追加するだけ)

### サンプル

```php
<?php

$query = auth()->user()
        ->traces()
        ->when(
            $this->search,
            fn ($query) =>
                $query->where(function ($query) {
                    $query->where('title', 'like', "%{$this->search}%")
                          ->orwhere('summary', 'like', "%{$this->search}%");
                })
        )
        ->when(
            $this->status,
            fn ($query) =>
                $query->where('status', $this->status)
        )
        ->when(
            $this->selectedTagId,
            fn ($query) =>
                $query->whereHas(
                    'tags',
                    fn ($q) => $q->where('tags.id', $this->selectedTagId)
                )
        );

switch ($this->sort) {
    case 'oldest':
        $query->oldest();
        break;
    case 'title':
        $query->orderBy('title');
        break;
    default:
        $query->latest();
}
```
