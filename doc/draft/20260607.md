# クエリビルダ

## when()

- 第1引数が`true`(データがある)のときだけ、第2引数の処理を実行
- `when($foo, fn ($query) => ...`

## where()

- 渡す引数によって2つの役割を行う

1. 通常の`where`(条件を追加するだけ)

引数に、`カラム名`, `演算子`, `値`の3つ(または2つ)を渡す。

```php
<?php
$query->where('title', 'like', '%検索%')
```

=> 生成されるSQL: `AND title LIKE '%検索%'` (ただ条件が横に並ぶだけ)

2. 関数を渡す`where`(カッコで囲む)

引数にカラム名でなく、`fn ($query) => ...`(関数)を丸ごと1つ渡す。

```php
<?php
$query->where(fn($q) => $q->where('title', '...')->orWhere('summary', '...'))
```

### サンプル

```php
<?php

$query = auth()->user()
        ->traces()
        ->when(
            $this->search,
            fn ($query) =>
                $query->where(function ($query) {
                    $query->where('title', 'like', "%{$this->search}%")
                          ->orwhere('summary', 'like', "%{$this->search}%");
                })
        )
        ->when(
            $this->status,
            fn ($query) =>
                $query->where('status', $this->status)
        )
        ->when(
            $this->selectedTagId,
            fn ($query) =>
                $query->whereHas(
                    'tags',
                    fn ($q) => $q->where('tags.id', $this->selectedTagId)
                )
        );

switch ($this->sort) {
    case 'oldest':
        $query->oldest();
        break;
    case 'title':
        $query->orderBy('title');
        break;
    default:
        $query->latest();
}
```
