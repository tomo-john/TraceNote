# アプリのテーマ

## 概要

作成するアプリのテーマなどメモ

### テーマ

```
「新しい領域を学習していく過程」を支援するアプリ」
```

> App名: TraceNote

### 理由

新しい製品知識を学習する中で、単なるメモではなく、理解の蓄積を整理したい。

新しい製品(やシステム)を学習する際、情報量が多く、知識同士のつながりを整理しながら学習する必要性を感じた。

TraceNote は、学習した内容を記録するだけでなく、「調査」「理解」「実践」といった学習状態を整理し、

知識を繋げながら蓄積していくことを目的としたアプリ。
