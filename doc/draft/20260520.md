# アロー関数と無名関数

### アロー関数

`fn() => ...`

特徴:

- 1つの式しか書けない
- `return`は不要
- 外側の変数を自動で使える(`use`不要)
- 短く書ける

```php
<?php
collect([[1, 2, 3]])
    ->map(fn ($n) => $n * 2);
```

=> `$n`には、1, 2, 3が順番に代入される

ブロック`{}`の中に複数行は書けない:

```php
<?php
// NG
fn ($n) => {
    $value = $n * 2;
    return $value;
};
```

### 無名関数

`function() { ... }`

特徴:

- 複数行書ける
- 途中変数を作れる
- if文やforeeachも使える
- `return`が必要

```php
<?php
collect([1, 2, 3])
    ->map(function ($n) {
        $value = $n * 2;
        return $value;
    });
```

if文:

```php
<?php
->map(function ($n) {
    if ($n % 2 === 0) {
        return 'even';
    }

    return 'odd';
});
```

### 外側の変数の扱い

アロー関数は自動で使える:

```php
<?php
$tax = 1.1;

collect([100, 200])
    ->map(fn ($price) => $price * $tax);
```

無名関数は`use`が必要:

```php
<?php
$tax = 1.1;

collect([100, 200])
    ->map(function ($price) use ($tax) {
        return $price * $tax;
    });
```

### メモ

- `return`だけならアロー関数
- 1行で終わるならアロー関数
- 複数行、変数、if、foreachが出たら無名関数
- まずアロー関数で書いてみる => 長くなってきたら無名関数に変える
