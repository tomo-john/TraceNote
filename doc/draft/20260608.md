# Enum(列挙型)

## 特定の決まった値のグループを安全かつスマートに管理する仕組み。

=> PHP 8.1から追加された仕様。

作成:

```
php artisan make:enum PostStatus
```

=> `app/Enums/PostStatus.php`が生成

編集:

```php
<?php
namespace App\Enums;

// : string をつけることで「Backed Enum」(値を持つEnum)になる
enum PostStatus: string
{
    case Draft = 'draft';
    case Published = 'published';
    case Archived = 'archived';
}
```

### モデルでキャスト

```php
<?php

use App\Enums\PostStatus;
use Illuminate\Database\Eloquent\Model;

class Post extends Model
{
    protected $casts = [
        // statusカラムをPostStatus Enumにキャストする
        'status' => PostStatus::class,
    ];
}
```

### 使い方サンプル

```php
<?php
// データの保存
$post = new Post();
$post->status = PostStatus::Published; // Enumのケースをそのまま代入
$post->save();

// データの取得と条件分岐
$post = Post::find(1);

if ($post->status === PostStatus::Draft) {
    // 下書きの場合の処理
}

// データベースに保存されている生の値を表示したい場合
echo $post->status->value; // 出力: "published"
```

### バリデーションでの利用

```php
<?php
use App\Enums\PostStatus;
use Illuminate\Validation\Rules\Enum;

$request->validate([
    // PostStatusに定義されている 'draft', 'published', 'archived' 以外は弾く
    'status' => [new Enum(PostStatus::class)],
]);
```

### Bladeでの利用

```blade
{{-- 条件分岐 --}}
@if($post->status === \App\Enums\PostStatus::Draft)
    <span class="badge-gray">下書き</span>
@endif

{{-- セレクトボックスの選択肢をループで作る --}}
<select name="status">
    @foreach(\App\Enums\PostStatus::cases() as $status)
        <option value="{{ $status->value }}">{{ $status->name }}</option>
    @endforeach
</select>
```

=> `PostStatus::cases()`を使うと、定義されているすべての`case`を配列として取得できる

=> `$status->value`は draft, publishedなどの割り当てられた値(右側)

=> `$status->name`は Draft, Publishedなどのコード上の名前(左側)

=> 専用の日本語ラベルなどを持たせたいときは、専用のメソッドなどを作る

```php
<?php

namespace App\Enums;

enum PostStatus: string
{
    case Draft = 'draft';
    case Published = 'published';
    case Archived = 'archived';

    // 日本語表示用のメソッドを追加
    public function label(): string
    {
        return match($this) {
            self::Draft => '下書き',
            self::Published => '公開中',
            self::Archived => 'アーカイブ済',
        };
    }
}
```

```php
<?php
echo $post->status->label(); // 出力: "公開中"
```
