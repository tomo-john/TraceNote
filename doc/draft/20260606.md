# ページネーション

## クラスにトレイトを持たせる

```php
<?php
use Livewire\WithPagination; // ① これを追加！

class Index extends Component
{
    // ② クラス内でトレイトを使う宣言をする
    use WithPagination;
....
```
- use Livewire\WithPagination; はトレイトを読み込む
- use WithPagination; はクラスでその機能を利用する宣言

## Computedへ

ページネーションするなら毎回DBから取得した方いい(必要な分だけを取得する)

全件取得して分割ではなく、必要な件数だけを取得してくれるのでComputedとの相性が良い。

```php
<?php
#[Computed]
public function traces()
{
    return auth()->user()
        ->traces()
        ->latest()
        ->paginate(6); // gird-cols-1 sm:grid-cols-2 lg:grid-cols-3 なので一旦6
}
```

- paginate(6) は1ページあたり6件表示
- SQLには LIMIT が付くため、必要なデータだけ取得される
- Collection ではなく LengthAwarePaginator が返る

## ページリンクの追加

```blade
<div class="mt-8">
    {{ $this->traces->links() }}
</div>
```

- `links()` がページ番号や「前へ」「次へ」を生成する
- ailwind CSSのデザインが自動で適用される

## 検索条件が変わったらページを1に戻す

```php
<?php
public string $search = '';

public function updatedSearch(): void
{
    $this->resetPage();
}
```
2ページ目にいる状態で検索すると、

- 検索結果は1ページしかない
- しかし現在ページは2ページ目

となり、データが存在しないページを表示してしまうことがある。

なので、`$this->resetPage();`で1ページ目に戻すのが定番。
